### Прототип комерческого приложения по учету автомобильной и техники специального назначения. 
* Для установки программы требуется:
    
    * Выполнить клонирование репозитория
     ```bash
     git clone https://github.com/kolesnikovvitaliy/pet_project_uavto.git
     ``` 
    * Перейти в директорию pet_project_uavto
    ```bash
     cd pet_project_uavto
     cd uavto
     ``` 
    * Запустить исполняемый файл
    ```bash
     ./uavto
     ``` 
* Для редактирования программы:
  
  * Выполнить клонирование репозитория
     ```bash
     git clone https://github.com/kolesnikovvitaliy/pet_project_uavto.git
     ``` 
   * Перейти в директорию pet_project_uavto
    ```bash
     cd pet_project_uavto
     ``` 
   * Создать и загрузить виртуальное окружение
    ```python
     python -m vevm env
     source ./env/bin/activate .
     ``` 
   * Установить зависимости
    ```python
    pip install -r requirements.txt
     ``` 
   * Перейти в директорию uavto_dev
    ```bash
     cd uavto_dev
     ``` 
   * Активировать приложение 
    ```python
     python uavto.py
     ```
